From quick sketch to floor plan, 3D model, and rough cost.

This demo shows a two-step workflow for generating architectural layouts from a sketch.

Sketch‑to‑Plan

Draw rectangles/circles to indicate rooms.

Auto 3D

Extrudes the generated plan into a simple 3D model and IFC file for quick visualization.

Quick Cost

Computes a rough cost based on RSMeans data using location, building type, and storey count.

How it works

Step 1 — Create a sketch in the Sketch Editor

Use the built‑in editor to place simple shapes that represent rooms and circulation. You can adjust sizes and positions until it matches your intent.

  • Draw basic room shapes (circles/rectangles)
  • Arrange and resize to suggest adjacency
  • Export the sketch when you’re ready
Clip: Creating a sketch with the editor.

Step 2 — Generate plan, 3D model, and estimate cost

Feed the sketch to the generator. It produces a floor plan, builds a simple 3D massing model, and estimates cost using high‑level inputs.

  • Preview a 3D model
  • Enter a few inputs for a rough cost estimate
